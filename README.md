# WebLogin

Projeto de um sistema web de cadastro e validações de login e cadastro de pessoas.

## Tecnologias Utilizadas

Para a construção deste sistema foram utilizadas as tecnologias como JavaEE, 
JavaScript, CSS, MySql e o Apache Tomcat. Também foram utilizadas as ferramentas 
MySql Workbench e a versão EE do Eclipse.

## Utilização

É necessário possuir uma versão Java JDK 1.8 ou superior. Ao utilizar este código fonte 
também é recomendado importá-lo para um projeto EE do Eclipse.
