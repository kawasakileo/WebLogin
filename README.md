# WebLogin

Projeto de um sistema web de cadastro e validações de login e cadastro de pessoas.

## Tecnologias Utilizadas

Para a construção deste sistema foram utilizadas as tecnologias como JavaEE, JSP, 
JavaScript, CSS, MySql e o Apache Tomcat. Também foram utilizados o MySql Workbench a versão EE do 
Eclipse IDE.

## Utilização

É necessário possuir uma versão Java JDK 1.8 ou superior. Ao utilizar este código fonte 
também é recomendado importá-lo para um projeto EE do Eclipse.
